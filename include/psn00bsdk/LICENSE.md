# PSn00bSDK Header Subset

This directory contains a small vendored subset of PSn00bSDK headers used by
CTR Native for PS1/Psy-Q-compatible types, constants, and inline helpers.

Source: <https://github.com/Lameguy64/PSn00bSDK>
